# Импорт из Excel (пользовательская инструкция)

Виджет `Импорт из Excel` загружает данные из файла и создает/обновляет сущности в amoCRM:

- сделки
- контакты
- компании

## 1. Что подготовить заранее

- Активный виджет `Импорт из Excel`.
- Синхронизированные поля amoCRM (кнопка синхронизации на странице виджета).
- Файл в формате `.xlsx`, `.xls` или `.csv`, размер до 10 МБ.
- Заголовки колонок в первой строке файла.

Важно:

- Обрабатывается только первый лист Excel-файла.
- Названия колонок в маппинге должны совпадать с заголовками в файле (включая регистр и пробелы).

## 2. Настройка виджета

Откройте: `Интеграции -> Импорт из Excel`.

Заполните блоки:

1. `Стандартные параметры`

- `Статус по умолчанию` (рекомендуется обязательно)
- `Ответственный по умолчанию`
- `Бюджет` (если нужен)
- `Название контакта`, `Название компании`, `Название сделки` (какую колонку использовать как имя)

2. `Соотношение полей`

- В `Поля сделки`, `Поля контакта`, `Поля компании` добавьте пары:
    - `Столбец Excel`
    - `Поле в amoCRM`

3. `Загрузка файла`

- Загрузите файл в поле `Excel файл`.

4. Нажмите `Сохранить`.

## 3. Запуск импорта

Процесс состоит из двух этапов.

1. На странице настроек нажмите `Начать импорт`.

- Файл ставится в очередь на разбор.
- После разбора строки появятся в `История импорта`.

2. Перейдите в `История импорта` и запустите выгрузку в amoCRM:

- `Выгрузить все` — отправить все строки, которые еще не завершены.
- или выделите нужные строки и нажмите `Запустить выгрузку`.

## 4. Статусы и результат

В `История импорта` используйте колонку `Статус`:

- `В процессе` / `Ждет выгрузки` — строка подготовлена или в очереди
- `Завершен` / `Успешно выгружен` — строка выгружена в amoCRM
- `Ошибка` / `Ошибка выгрузки` — строка не выгружена

После успешной выгрузки появляются ссылки на созданные/обновленные:

- сделку
- контакт
- компанию

## 5. Как работает проверка дублей

Если в маппинге контактов/компаний есть телефон или email, виджет сначала пытается найти существующую сущность в amoCRM:

- нашел — обновляет найденную
- не нашел — создает новую

Чтобы проверка дублей работала лучше:

- обязательно маппите телефон и/или email
- используйте единый формат телефонов и email в файле

## 6. Типовые ошибки и что делать

1. `Настройки не активны или не найдены`

- Проверьте, что виджет активен.

2. `Выберите файл для импорта`

- Файл не загружен в настройках виджета.

3. Строки не уходят в amoCRM

- Проверьте, что после разбора вы запустили `Выгрузить все` или `Запустить выгрузку`.
- Проверьте корректность маппинга полей.
- Проверьте, что выбран статус по умолчанию.

4. Много дублей

- Проверьте маппинг телефонов/email.
- Не нажимайте запуск выгрузки повторно без необходимости.

## 7. Рекомендованный порядок работы

1. Синхронизировать поля amoCRM.
2. Заполнить стандартные параметры.
3. Настроить маппинг полей.
4. Загрузить файл.
5. Нажать `Начать импорт`.
6. В истории нажать `Выгрузить все`.
7. Проверить `Ошибки выгрузки` и при необходимости перезапустить выборочно.
